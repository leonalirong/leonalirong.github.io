app.use 与 app.all 有什么区别

  1. app.all 其实可以理解为 是 app.get app.post app.delete …. 的一个简写方式
  2. app.use 是使用中间件的。
  3. app.all 里面的 PATH 路径是全匹配。 app.use 里面的 PATH 是一个前缀

2 .模板引擎有很多种，duge、、

3.使用

1.安装 ejs

2.在 express 代码设置使用什么模板引擎，使用哪个文件夹做为模板页面的文件夹

3.在路由的回调函数中，使用res.render (‘模板页面的文件名字’, 需要带到模板页面中的数据) 渲染页面。
