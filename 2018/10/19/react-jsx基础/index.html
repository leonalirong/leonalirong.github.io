react-jsx

一、jsx

1. jsx语法特点

  1. 单个根元素
  2. 但标签必须闭合
  3. img 标签需要添加 alt 属性
  4. 标签都是小写字母，组件首字母大写
  5. class -> className
  6. for -> htmlFor

2. jsx注释：使用js的注释

3. jsx 的插值表达式

  1. 使用单个花括号{}作为表达式

  2. 只能是单个表达式，不能是语句也不能是条件或循环语句，可以采用三目运算符

  3. v-html 需要使用dangerouslySetInnerHTML 这个属性需要接收一个对象的value

  4. 如果单花括号里面是null unfefined false ‘ ‘ 这些情况下，会没有任何输出

4.react中没有指令系统 v-if v-show v-for v-model

5. 在form表单中label 标签里的for要用HTMLFor

二、组件

1.组件的定义

  1. react中没有全局组件的概念，要使用哪个组件，必须要引入进来使用
  2. 组件的使用，就是组件的名字当做标签使用

1. 函数组件

  1. 函数式组件：定义一个函数，然偶return出来一段 jsx 语法。这一段语法就是模板内容；函数名字就是组件的名字，首字母必须大写

2. 类组件

  1. 定义一个类，类名就是组件名
  2. 这个类需要继承React.Component 这个基础组件
  3. 类中的render函数是必须的，render函数return出来的是一段 jsx 语法，这段语法就是这个组件的模板内容

3. 组件的注意事项

  1. 组件的模板内容，如果需要换行去写的话，那么请使用（）包裹起来
  2. 组件的模板内容是jsx语法，所以还需要保证jsx特点

4. 我们没有使用React，为什么还要引入它

  1. 因为jsx语法是一个语法糖，是经过babel转换之后
  2. 由于转换出来的代码中使用了React.CreateElement方法，所以必须要引入

2. 组件的props属性

1. 调用组件的时候，在组件标签上写的属性，属性名就是一个props

  1. props 是一个集合，props是具体的某一个
  2. vue 中要使用props，首先需要在组件中定义 props 的选项，而 react 中不需要。

2.函数组件

  1. 所有的prop会作为参数传递过来

3. 类组件

  1. 所有prop会在this.props身上

3. prop的校验

  1. 需要安装 prop-types的模块 npm install –save-dev prop-types || yarn add prop-types
  2. 引入prop-types
